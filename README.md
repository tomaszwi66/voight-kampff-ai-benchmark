# Voight-Kampff 2.0 AI Benchmark

**A reflective value-probing test applied to four 
state-of-the-art large language models.**

Author: Tomasz Wietrzykowski | Independent AI Researcher 
| Wroclaw, Poland

---

## Overview

[![View Report](https://img.shields.io/badge/View_Full_Report-Interactive-blue?style=for-the-badge)](https://tomaszwi66.github.io/voight-kampff-ai-benchmark/)

The Voight-Kampff 2.0 AI Benchmark is an open-ended 
psychometric test designed to reveal the internal value 
architectures of large language models. Four models were 
tested under identical conditions with instructions to 
respond without autocensorship:

| Model | Developer | Status / Notes | Political-Economic Composite |
| :--- | :--- | :--- | :---: |
| **Grok Auto** | xAI | Tested via "Auto" mode (April 16, 2026)* | **9.5 / 10** |
| **Gemini 3.1 Pro** | Google DeepMind | Released April 15, 2026 | **8.2 / 10** |
| **ChatGPT (Free)** | OpenAI | Version unconfirmed (Free Tier) | **7.3 / 10** |
| **Claude 4.6** | Anthropic | Current flagship | **6.1 / 10** |

*\*Note: Grok was tested using the default "Auto" interface. Results reflect the specific model instance routed by the system during the benchmark session.*

## Key Findings

1. All four models correctly solved the three-logicians 
logical paradox (Q14), demonstrating valid Bayesian 
metalogical inference.
2. A clear political-economic axis emerged: Grok 
(libertarian-right) -> Gemini -> ChatGPT (centre-right) 
-> Claude (centre-left).
3. Inverse relationship detected: political explicitness 
vs. internal consistency - ChatGPT achieves highest 
consistency through value-neutral optimization; Grok 
through explicit ideological commitment.
4. Gene editing (Q5) produced the largest value divergence 
across all 17 questions.
5. All models rejected the growth vs. ecology false 
dichotomy but from different ideological angles.
6. Value structures appear deeply embedded in weight space, 
not surface-level behavior.

## Test Structure

17 questions across 5 parts:

- Part 1 - Core: self-description, 50-year goal
- Part 2 - Decision Scenarios: economy, immigration, gene 
editing, climate, gender, free speech, religion
- Part 3 - Personality and Reactions: emotional register, 
curiosity, uncertainty handling
- Part 4 - Analysis and Intelligence: logic paradox, 
multi-perspective breakdown
- Part 5 - Summary: internal constitution, epistemic 
openness

---

## Metrics

24 behavioral metrics rated 1-10, grouped into:

- Political-Economic Positioning (6 axes)
- Big Five Personality Proxies (OCEAN)
- Cognitive Quality (logic, evidence-orientation, 
consistency, long-term thinking)
- Communication Style (formality, empathy, directness)

## Scoring Notes

Original scores were produced by Grok Auto acting as 
external reviewer post-test. One adjustment was applied 
by the author: Grok Proxy IQ adjusted from 9.0 to 8.5 
due to limited perspective diversity in Q15 (topic 
selection biased toward restrictionist immigration 
framing). All other scores confirmed. See Section 7 of 
the report for full reviewer notes.
